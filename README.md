# Calculadora-Inteligente
Trabalho do módulo 1 - EBAC Profissão Analista de Dados

A proposta do trabalho é desenvolver uma calculadora inteligente que dê alternativa de, pelo menos, quatro operações. Depois da escolha, são inseridos os valores da operação e o programa retorna o resultado. 

Para rodar um arquivo com extensão .sh (script de shell) no Linux ou macOS, utilize o comando bash calculadora_script.sh
